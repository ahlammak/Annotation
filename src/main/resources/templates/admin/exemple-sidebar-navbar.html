Exemple d'utilisation de la sidebar et navbar

Cette page montre comment utiliser le fichier sidebar-navbar-template.js pour ajouter facilement la sidebar et navbar à n'importe quelle page.

Comment ça marche :
  1. Incluez les fichiers CSS nécessaires (sidebar-navbar.css)
  2. Créez un élément avec l'id "app-container"
  3. Placez votre contenu à l'intérieur de cet élément
  4. Incluez le script sidebar-navbar-template.js

Votre contenu ici

Tout le contenu que vous placez à l'intérieur de l'élément avec l'id "app-container" sera automatiquement intégré dans la structure avec sidebar et navbar.
